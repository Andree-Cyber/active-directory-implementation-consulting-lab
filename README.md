# Associate Consultant Intern Labs

This repository contains hands-on IT consulting and cybersecurity lab projects designed to demonstrate technical implementation, documentation, and client-focused problem solving.

## Projects

### 01 - Active Directory Implementation

This project simulates an Active Directory implementation for a small business client. The lab includes Windows Server 2022, Active Directory Domain Services, domain user accounts, security groups, shared folder permissions, Group Policy, and a domain-joined Windows 10 workstation.

## Skills Demonstrated

- Windows Server administration
- Active Directory Domain Services
- User and group management
- Group Policy configuration
- Shared folder permissions
- Windows client domain join
- IT consulting documentation
- Client modernization planning
